## ⚠️ 중요: 세그먼트 특성 정확한 이해

### 🔍 Segment 0, 1 = 이탈 위험 / 휴면 고객 (희귀 0.046%)

**잘못된 이해**: VIP 고가치 고객  
**올바른 이해**: 과거 활동했으나 현재 이탈 위험이 있는 고한도 보유 고객

**데이터 기반 특징**:
- 높은 신용한도 (300만원 이상) 보유
- 하지만 12개월 사용액 10만원 미만
- 최근 3~12개월 사용 없음 (휴면 상태)
- CA(현금서비스) 최종 이용 후 1000일 이상 경과
- 과거엔 해외 사용/백화점 등 고액 사용 기록 있음
- 현재는 **"잠자는 고한도 카드"** 상태

**핵심 인사이트**:
```
v4 모델이 탐지하는 것 = "고한도 보유 + 장기 비활성"
→ 재활성화하면 수익 잠재력 높은 고객
→ 방치하면 해지/타사 이동 위험
```

### 📊 Segment 2, 3, 4 = 활동 고객 (일반 99.954%)

**Segment 2 (5.3%)**: 중상위 활동, 안정적 사용  
**Segment 3 (14.6%)**: 중위 활동, 변동 있음  
**Segment 4 (80.1%)**: 소액 고빈도 일상 사용
